# MVP - Machine Learning


## 📌 Objetivo
Este projeto tem como objetivo desenvolver um modelo de Machine Learning capaz de prever a probabilidade de fechamento de oportunidades comerciais com base em dados de CRM.

 
## 📊 Dataset
A base utilizada contém dados de pipeline de vendas, incluindo informações sobre oportunidades, valores, estágios e datas.

 
## 🎯 Problema de negócio
A proposta é apoiar a tomada de decisão comercial, permitindo:
- priorização de oportunidades
- estimativa de receita esperada
- análise do funil de vendas
 

## 🤖 Modelos
Foram testados modelos de classificação para prever o sucesso das oportunidades.

 
## 🚀 Tecnologias
- Python
- Pandas
- Scikit-learn
- Google Colab

 
## 📁 Estrutura
- dataset: dados utilizados (sales_pipeline.csv) 
- notebook: desenvolvimento do modelo de Machine Learning
